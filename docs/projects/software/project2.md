# Apocalypse: You Only Die Once

## Overview
Apocalypse is a 2-player, first-person zombie shooter & survival game. Developed with **Unity** and **C#**, and deployed with **Vercel**.

_This game was submitted as a group final project for the MIT course 6.4570: Creating Video Games._
